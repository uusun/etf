接口自检与部署说明

这个页面如何获取净值？

页面优先通过 https://fund.eastmoney.com/pingzhongdata/{基金代码}.js 读取历史单位净值；盘中估算模式会再尝试通过 https://fundgz.1234567.com.cn/js/{基金代码}.js 读取估算净值。GitHub Pages 是纯静态网页，所以采用 JSONP / script 标签方式规避浏览器 CORS 限制。

如果某个接口失败，该基金会自动退回到本文件内置的基准净值与市值，并在“接口状态”列标记。

上传 GitHub Pages 的最简单方法
  1. 新建或打开你的 GitHub Pages 仓库。
  2. 把本文件重命名为 index.html 并上传到仓库根目录。
  3. 进入 Settings → Pages → Branch 选择 main / root。
  4. 打开 GitHub Pages 给出的地址。

注意：如果仓库或 Pages 是公开的，基金代码、份额、成本和市值会被公开看到。
